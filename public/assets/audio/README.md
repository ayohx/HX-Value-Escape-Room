This directory contains placeholder references for sound effects.

## Sound Files

- `success.wav` - Success/completion sound
- `fail.wav` - Failure/error sound
- `click.mp3` - UI click sound
- `unlock.wav` - Room unlock/achievement sound

## Note

These are placeholder references. Replace with actual audio files when available.

## Audio Requirements

- Format: WAV or MP3
- Size: < 100KB per file
- Duration: 0.5-2 seconds
- Volume: Normalized to -3dB peak
